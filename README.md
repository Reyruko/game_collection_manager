                                            Game Collection Manager

--------------------------------------------------------------------------------------------------------------

A web application for managing personal game collections, tracking achievements, and monitoring gaming progress.

Built as an educational project during the Spring Fundamentals course and designed to be extended further during Spring Advanced.

--------------------------------------------------------------------------------------------------------------

                                                    Overview

Game Collection Manager allows users to organize their personal gaming library in one place.

Users will be able to register, log in, add games to their collection, track playtime, rate games, manage achievements, and monitor progress.

The goal of the project is to combine practical Spring development concepts with a domain centered around gaming and collection management.
